# Ticket: UI/UX ブラッシュアップ & アニメーション

**ID**: T-006-UI-Polish
**Status**: Pending

## 目標 (Goal)
MVPとしての完成度を高めるため、マイクロインタラクション、ローディング表現、モバイル対応を徹底する。

## 要件 (Requirements)
- **Animations**: `framer-motion` を使用したスムーズな遷移。
- **Loading States**: 
    - AI生成中の待機時間を「退屈させない」時間にする（AI店主が考えている様子など）。
    - 3Dローディングの統合。
- **Mobile Responsive**: スマホでの操作性（タップ領域、フォントサイズ）の最適化。

## 実装手順 (Implementation Steps)
1. [ ] **アニメーション導入**:
    - ページ遷移時のフェードイン/アウト。
    - リスト表示時のStagger効果（カードが順番に表示される）。
2. [ ] **ローディング演出**:
    - 「老舗の歴史を紐解いています...」等のメッセージ表示。
    - 3D暖簾のゆらめき速度変化など。
3. [ ] **モバイル実機確認・修正**:
    - 余白（Padding/Margin）の微調整。
    - タッチターゲットの確保（44px以上）。

## 検証 (Verification)
- スマホ実機（またはシミュレータ）で触り心地が良いか。
- ユーザーを待たせる場面でストレスがないか。
