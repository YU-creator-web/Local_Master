# Ticket: 金箔パーティクル演出の強化

**ID**: T-013-Particles
**Status**: ✅ 完了

## 目標 (Goal)
ランディングページ、検索画面、詳細画面で統一された、リッチで美しい「舞う金箔」エフェクトを実装する。
特に、サイズにばらつき（大・中・小）を持たせ、四角い金箔が優雅に舞う様子を表現する。

## 要件 (Requirements)
1.  **パーティクル表現の強化 (`src/components/canvas/Particles.tsx`)**:
    *   形状: 四角形 (PlaneGeometry or custom shader equivalent).
    *   サイズ: ばらつきを持たせる (小・中・大)。
    *   色: リッチな金色 (#D4AF37 ベースで輝きを持たせる)。
    *   動き: 優雅に舞う (重力だけでなく、少し揺らぎを入れる)。
2.  **全画面への統一適用**:
    *   Landing (`/`)
    *   Search (`/search`)
    *   Detail (`/shop/[id]`)
    *   これら全ての背景として統一されたエフェクトを配置する。

## 変更対象ファイル
- `src/components/canvas/Particles.tsx`: ロジック改修。
- `src/app/layout.tsx` (必要であれば): グローバルに配置するか、各ページで呼び出すか検討。現状は各ページで呼んでいる可能性が高い。
